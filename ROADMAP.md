## ⬜ В очереди (новые фичи)

| Задача | Описание | Сложность |
|--------|----------|-----------|
| Сортировка каналов и групп | Drag-and-drop (`@dnd-kit`) или диалог «Упорядочить» со списком. `sort_order` в БД уже есть. На мобильных кнопки ↑↓ предпочтительнее. | ⭐⭐⭐ |
| Иконка приложения (favicon / PWA) | favicon.ico + 192×192 и 512×512 PNG для PWA manifest. | ⭐ |
| Pull-to-refresh на мобильных | Overscroll → refresh канала. Не конфликтовать с нативным pull-to-refresh браузера. | ⭐⭐⭐ |
| Минимизировать хедеры на мобильных | (A) AppHeader скрывается при скролле; (B) NewsFeedToolbar схлопывается; (C) компактный мобильный тулбар. | ⭐⭐⭐ |
| Сквозной медиа-просмотр | Лайтбокс по клику на фото: навигация по всем новостям канала, ←→/Esc. Медиа-каналы: авто-прочитано. | ⭐⭐⭐⭐ |
| Шаринг новости | Web Share API на мобильных; clipboard fallback на десктопе. | ⭐⭐ |
| Полная клавиатурная навигация (zero-mouse) | Space листает новости → refresh → автопереход на следующий канал с непрочитанными → по кругу. | ⭐⭐⭐ |
